#GitHub Commands
___
##status
+ git status
  
##branches you have created
+ git branch
  
##deleting the whole branch
+ git checkout master
  
+ git branch -D test
  
##add files before pushing
+ git add test
  
##commit files
+ git commit test
  
+ git commit -a test

+ git commit -m "message" test

##remove deleted files
+ git commit (deleted) test

##pull & push
+ git push origin master
  
+ git pull origin master
